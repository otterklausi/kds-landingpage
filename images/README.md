# Bilder Ordner

Hier können alle Bilder für die Landing Page abgelegt werden.

## Empfohlene Bilder

### Header/Hero Bereich
- **hero-bg.jpg** - Großes Hintergrundbild für den Hero-Bereich (1920x1080px empfohlen)
- **hero-worker.jpg** - Bild eines Handwerkers bei der Arbeit

### Portfolio/Referenzen
- **project-1.jpg** - Kellerabdichtung Projekt
- **project-2.jpg** - Dachsanierung Projekt
- **project-3.jpg** - Fassadensanierung Projekt

### Icons/Logos
- **logo.png** oder **logo.svg** - Firmenlogo (optional, falls SVG nicht verwendet wird)

## Bildformate
- **.jpg** für Fotos
- **.png** für Bilder mit Transparenz
- **.webp** für moderne Browser (kleinere Dateigröße)
- **.svg** für Logos und Icons

## Optimierung
Bilder sollten vor dem Hochladen optimiert werden:
- Dateigröße: Max. 200KB pro Bild
- Abmessungen: Nicht größer als benötigt
- Kompression: 80-85% Qualität für JPG

## Tools zur Optimierung
- https://squoosh.app/ (Online)
- https://tinypng.com/ (Online)
- Photoshop "Für Web speichern"

## Verwendung im Code

```html
<img src="images/hero-bg.jpg" alt="KDS Handwerker bei der Arbeit">
```

---

**Hinweis:** Aktuell werden Platzhalter verwendet. Ersetze diese durch echte Bilder für die finale Version.
